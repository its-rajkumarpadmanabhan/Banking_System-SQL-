🏦 Bank Management System

📌 Project Summary

Developed a Bank Management System using SQL to efficiently manage banking operations, including customer accounts, loans, branch information, transactions, and financial records. The system enables accurate tracking of due dates, payments received, outstanding balances, and customer data through a well-structured relational database. It is designed to support future enhancements such as interest calculations, payment reminders, and financial reporting.
